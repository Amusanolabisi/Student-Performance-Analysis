🎓 Student Performance Analysis

An end-to-end data analysis project exploring academic outcomes across 50,000 students and 100 schools in Lagos — built with SQL, Python, Excel, and Power BI.


📌 Project Overview

This project analyzes student performance data across 50,000 students, 100 schools, and multiple subjects and terms in Lagos State. The goal was to uncover what's really driving student outcomes and give educators, school administrators, and policymakers a clear, filterable view of performance across LGAs, school types, genders, and subjects.

🔍 Key Insights

✅ Overall pass rate stands at 86.15% but an average score of 63.01 reveals many students are barely passing, not excelling

📅 Attendance correlates strongly with scores — visible clearly in the scatter plot analysis

📚 English leads all subjects in pass rate across all three terms

🏫 Performance remains consistent across terms, suggesting systemic rather than seasonal factors at play

📍 Significant variation exists across LGAs — pointing to inequality in educational outcomes across Lagos


✨ Special Features — Bookmark Toggles
The dashboard includes two interactive bookmark toggles designed to give users a focused view depending on what they need:

🏫 School View — switches the dashboard to a school-level perspective, allowing stakeholders to drill into individual school performance

📚 Subject View — switches to a subject-level breakdown, making it easy to compare performance across all subjects at a glance

These toggles were intentionally built to reduce dashboard clutter and improve the user experience — giving different stakeholders exactly the view they need without overwhelming them with information.

🛠️ Tools Used

SQL- Data extraction and querying

Python(Jupyter Notebook)- Data cleaning and exploratory analysis

Excel- Data validation and modelling

Power BI- Dashboard, visualization, and bookmark toggles
